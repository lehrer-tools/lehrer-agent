# Didaktik Physik

Allgemeine fachdidaktische Referenz für den Physikunterricht (Sekundarstufe I).

## Notation und Konventionen

### Physikalische Größen und Einheiten
| Größe | Formelzeichen | Einheit | Symbol |
|-------|---------------|---------|--------|
| Geschwindigkeit | v | Meter pro Sekunde | m/s |
| Kraft | F | Newton | N |
| Masse | m | Kilogramm | kg |
| Energie | E | Joule | J |
| Arbeit | W | Joule | J |
| Leistung | P | Watt | W |
| Spannung | U | Volt | V |
| Stromstärke | I | Ampere | A |
| Widerstand | R | Ohm | Ω |
| Temperatur | T / ϑ | Kelvin / Grad Celsius | K / °C |
| Frequenz | f | Hertz | Hz |

### Formeln (Grundformeln Kl. 7–10)
| Thema | Formel | Klasse |
|-------|--------|--------|
| Geschwindigkeit | v = s / t | 7–8 |
| Kraft/Gewichtskraft | F = m · g | 7–8 |
| Arbeit | W = F · s | 8–9 |
| Leistung | P = W / t | 8–9 |
| Ohmsches Gesetz | U = R · I | 8–9 |
| Kinetische Energie | E_kin = ½ · m · v² | 9–10 |
| Potentielle Energie | E_pot = m · g · h | 9–10 |
| Wirkungsgrad | η = E_nutz / E_zu | 9–10 |

### Formelumstellung (Dreieck-Methode)
Für A-Kurs bewährt: Formeldreieck zum mechanischen Umstellen.
Beispiel Ohmsches Gesetz:
```
    ┌─────┐
    │  U  │
    ├──┬──┤
    │ R│ I│
    └──┴──┘
  U = R · I    R = U / I    I = U / R
```

### Protokollformat
| Abschnitt | Inhalt |
|-----------|--------|
| **Thema** | Titel des Experiments |
| **Frage/Hypothese** | Was untersuchen wir? Was vermuten wir? |
| **Material** | Liste mit Bildern/Skizzen der Geräte |
| **Durchführung** | Nummerierte Schritte mit Skizzen |
| **Beobachtung** | Was siehst du? (Tabelle, Skizze, Foto) |
| **Auswertung** | Erklärung, Formel, Berechnung |
| **Ergebnis** | Merksatz / Antwort auf die Frage |

## Typische Stundenstruktur

### Einführung (ca. 10 Min)

1. **Phänomen zeigen**: Demonstrationsexperiment, Video,
   Bild oder Alltagsgegenstand als Einstieg
2. **Frage entwickeln**: „Was passiert hier? Warum?" — schriftlich an der Tafel
3. **Hypothesen sammeln**: Schüler schreiben Vermutungen auf Karten → an Tafel heften
4. **Begriffe vorentlasten**: Neue Fachbegriffe an der Tafel erklären (Wort + Bild + Einheit)

### Übung (ca. 25 Min)
1. **Experiment**: Schülerexperiment in Kleingruppen nach schriftlicher Anleitung
   (Arbeitsblatt mit Bildern, nummerierte Schritte, Sicherheitshinweise).
   Beobachtungsauftrag als Tabelle zum Ausfüllen.
2. **Auswertung**: Ergebnisse in Tabelle/Diagramm eintragen.
   Formel einführen und Beispielrechnung an der Tafel.
3. **Übungsaufgaben**: Formel anwenden, Einheiten umrechnen,
   Diagramme auswerten (Arbeitsblatt)

### Sicherung (ca. 10 Min)
1. **Ergebnisse vergleichen**: Tabellen/Werte der Gruppen an der Tafel sammeln
2. **Merksatz formulieren**: Gemeinsam erarbeiten, ins Heft übernehmen
3. **Formel sichern**: Formel + Einheiten + Beispiel ins Formelheft
4. **Hausaufgabe**: Berechnungsaufgaben oder Protokoll vervollständigen

## Differenzierung A-Kurs / B-Kurs

Die Fachleistungsdifferenzierung in Physik beginnt je nach Bundesland
ab Klasse 7 oder 9 (Physik oder Chemie).

### A-Kurs (grundlegende Bildung — EBR)
- **Experimente**: Stärker angeleitet; Beobachtungstabelle vorgegeben;
  Lückentext für Auswertung
- **Formeln**: Grundformeln einsetzen (Werte in Formel einsetzen);
  Formeldreieck als Hilfe; kein eigenständiges Umstellen
- **Berechnungen**: Einfache Einschritt-Rechnungen; Einheiten gegeben;
  Zahlenwerte „passend" (ganzzahlig, überschaubar)
- **Diagramme**: Werte aus Diagramm ablesen; einfache Zuordnungsaufgaben
- **Textaufgaben**: Kurz, klar, eine Frage; relevante Größen hervorgehoben
- **Aufgabenformate**: Lückentext, Zuordnung, Multiple Choice,
  Werte in Formel einsetzen

### B-Kurs (erweiterte Bildung — FOR)
- **Experimente**: Offenere Aufgabenstellung; eigene Beobachtungstabelle anlegen;
  Hypothese formulieren; Fehlerbetrachtung
- **Formeln**: Umstellen nach gesuchter Größe; mehrere Formeln kombinieren;
  Einheiten selbstständig umrechnen
- **Berechnungen**: Mehrstufige Rechnungen; Einheitenumrechnung nötig;
  realitätsnahe Zahlenwerte
- **Diagramme**: Diagramme selbst erstellen; Zusammenhänge beschreiben;
  Proportionalitäten erkennen
- **Textaufgaben**: Komplexer, mehrere Schritte; relevante von irrelevanten
  Informationen trennen; Alltagsbezug
- **Aufgabenformate**: Offene Fragen, Berechnungen mit Umstellung,
  Erklärung/Begründung, Vergleiche

### Differenzierungsbeispiel
```
Aufgabe: Ohmsches Gesetz

A-Kurs:
  An einem Widerstand liegt eine Spannung von U = 12 V an.
  Die Stromstärke beträgt I = 2 A.
  Berechne den Widerstand R.
  Formel: R = U / I = ___ / ___ = ___ Ω

B-Kurs:
  Eine Glühlampe hat einen Widerstand von R = 60 Ω.
  Berechne die Stromstärke, wenn die Lampe an 230 V
  angeschlossen wird.
  Zusatz: Berechne die elektrische Leistung der Lampe.
```

## Visuelle Methoden und Materialien

### Grundprinzip
Physikalische Phänomene sind oft unsichtbar (Strom, Kraft, Energie).
Sie müssen durch Modelle, Analogien und Visualisierungen sichtbar gemacht werden.

### Methoden
| Methode | Beschreibung | Einsatz |
|---------|-------------|---------|
| **Demonstrationsexperiment** | Lehrkraft zeigt Versuch, Schüler beobachten | Einstieg, Phänomen zeigen |
| **Schülerexperiment** | Kleingruppen experimentieren nach Anleitung | Kompetenzerwerb, Handlungsorientierung |
| **Schaltbilder / Schaltpläne** | Genormte Symbole für Stromkreise | Elektrizitätslehre — rein visuell |
| **Diagramme** | Messwerttabelle → Diagramm (Achsen beschriftet!) | Auswertung, Zusammenhänge erkennen |
| **Formelkarten** | Laminierte Karten: Formel + Einheiten + Beispiel | Nachschlagewerk, Prüfungshilfe |
| **Modelle / Analogien** | Wassermodell für Strom, Dominosteine für Energiekette | Abstrakte Konzepte veranschaulichen |
| **Stationenlernen** | Experimentstationen mit schriftlicher Anleitung | Selbstständiges Arbeiten, Differenzierung |
| **Piktogramme für Sicherheit** | GHS-Gefahrensymbole, Laborregeln als Bilder | Sicherheitsunterweisung visuell |
| **Farbcodierung** | Kabel: rot = Plus, blau = Minus; Temperaturen: rot = warm, blau = kalt | Schnelle visuelle Orientierung |
| **Realexperimente mit sichtbarer Wirkung** | Magnetfeldlinien mit Eisenspänen | Unsichtbares sichtbar machen |
| **Videos** | Zeitlupen, Simulationen, Dokumentationen | Phänomene zeigen, die im Klassenraum nicht möglich sind |
| **Simulationssoftware** | PhET-Simulationen (Stromkreise, Optik) | Interaktiv, visuell |
| **Dokumentenkamera** | Messgeräte live projizieren (Multimeter, Thermometer) | Alle können gleichzeitig ablesen |

### Tafelarbeit
- Stundenziel + Ablauf sichtbar
- Fachbegriffe: Wort — Symbol — Einheit — Bild (vier Spalten)
- Formeln groß und deutlich, Formeldreieck daneben
- Beispielrechnungen: Schritt für Schritt mit Einheiten
- Merksätze eingerahmt
