# Didaktik Deutsch

Allgemeine fachdidaktische Referenz für den Deutschunterricht (Sekundarstufe I).

## Notation und Konventionen

### Schriftliche Arbeiten

- **Kein Aufsatz unter Zeitdruck ohne Hilfsmittel** — Wörterbuch als Hilfsmittel erlaubt
- Grammatikaufgaben: Bestimmung (unterstreichen, einkreisen), Lückentext,
  Umformung, Zuordnung
- Textanalyse: Leitfragen, Textbelege markieren, Tabellen ausfüllen

### Korrektursymbole (am Rand)
| Zeichen | Bedeutung |
|---------|-----------|
| R | Rechtschreibfehler |
| Gr | Grammatikfehler |
| Z | Zeichensetzungsfehler |
| A | Ausdrucksfehler |
| W | Wortfehler / Wortwahl |
| Sb | Satzbaufehler |
| T | inhaltlicher/logischer Fehler |

### Operatoren (Anforderungsbereiche)
| AFB | Operatoren |
|-----|-----------|
| I (Reproduzieren) | nenne, gib an, beschreibe, ordne zu, markiere |
| II (Zusammenhänge) | vergleiche, erkläre, fasse zusammen, formuliere um |
| III (Reflektieren) | beurteile, nimm Stellung, begründe, interpretiere |

## Typische Stundenstruktur

### Einführung (ca. 10 Min)
1. **Einstieg visuell**: Bild, kurzer Textauszug, Zitat oder Gegenstand als Impuls
2. **Vorwissen aktivieren**: Mindmap an der Tafel, Brainstorming (schriftlich auf Karten)
3. **Stundenziel** an der Tafel visualisieren (z.B. „Heute: Inhaltsangabe schreiben")
4. **Schlüsselbegriffe vorentlasten**: Neue Fachbegriffe oder schwierige Wörter
   an der Tafel erklären, mit Beispiel und ggf. Bild

### Übung (ca. 25 Min)
1. **Textarbeit**: Gemeinsames Lesen (Lehrkraft liest vor,
   Schüler lesen mit). Schwierige Stellen klären.
   Verständnisfragen schriftlich beantworten.
2. **Sprachliche Übung**: Grammatikarbeitsblatt, Satzbauübung,
   Rechtschreibtraining (visuell: Lernkartei, Wortbilder)
3. **Schreibaufgabe**: Texte verfassen nach Vorlage/Muster;
   Schreibkonferenz mit Partnerkontrolle

### Sicherung (ca. 10 Min)
1. **Ergebnisse vergleichen**: Tafel, Dokumentenkamera, Schülerpräsentation
2. **Merksatz/Regel** ins Heft übernehmen (Grammatik-Kasten, Definition)
3. **Hausaufgabe** schriftlich an der Tafel

## Differenzierung A-Kurs / B-Kurs

Die Fachleistungsdifferenzierung in Deutsch beginnt je nach Bundesland
ab Klasse 7 oder 9. Vorbereitung ab Klasse 7/8 durch Binnendifferenzierung.

### A-Kurs (grundlegende Bildung — EBR)
- **Lesen**: Kürzere, sprachlich einfachere Texte; Leitfragen zum Textverständnis;
  Multiple Choice; Richtig/Falsch; Informationen aus dem Text entnehmen
- **Grammatik**: Wortarten bestimmen (Grundwortarten), einfache Satzglieder,
  Aktiv/Passiv (erkennen), Kommaregeln (Aufzählung, Nebensatz)
- **Schreiben**: Kürzere Texte (150–250 Wörter); geleitetes Schreiben mit
  Satzanfängen, Textbausteinen; Inhaltsangabe mit Leitfragen;
  Beschreibung (Vorgang, Person) nach Muster
- **Rechtschreibung**: Lückentexte, Fehlersuchaufgaben, Abschreibtraining
- **Textanalyse**: Einfache Sachtexte; Tabellen/Stichpunkte statt Fließtext

### B-Kurs (erweiterte Bildung — FOR)
- **Lesen**: Längere, komplexere Texte (auch literarisch); offene Verständnisfragen;
  Zusammenfassungen in eigenen Worten; Textvergleich
- **Grammatik**: Alle Wortarten + Attribute, Adverbialsätze, Konjunktiv I+II,
  Zeichensetzung bei Zitaten, Einschüben, indirekter Rede
- **Schreiben**: Längere Texte (300–500 Wörter); eigenständige Argumentation;
  Erörterung (linear Kl. 8, dialektisch Kl. 9–10); Interpretation;
  Bewerbungsschreiben + Lebenslauf
- **Rechtschreibung**: Diktatersatz mit Fehleranalyse und Regelanwendung
- **Textanalyse**: Sachtexte und literarische Texte; Stilmittel erkennen;
  Autorintention; Textsortenvergleich

### Differenzierungsbeispiel
```
Aufgabe: Inhaltsangabe zur Kurzgeschichte "Das Brot" (W. Borchert)

A-Kurs:
  1. Lies die Geschichte. Beantworte die Fragen in Stichworten:
     - Wer? Wo? Wann?
     - Was passiert?
     - Wie endet die Geschichte?
  2. Schreibe eine Inhaltsangabe (8–10 Sätze).
     Nutze die Satzanfänge:
     "In der Kurzgeschichte geht es um ..."
     "Die Handlung spielt ..."
     "Am Ende ..."

B-Kurs:
  1. Fasse den Inhalt in einer Inhaltsangabe zusammen (12–15 Sätze).
  2. Erkläre, warum die Frau so handelt, wie sie handelt.
     Belege deine Aussage mit Textstellen.
  3. Welche Bedeutung hat das Brot als Symbol? Begründe.
```

## Visuelle Methoden und Materialien

### Grundprinzip
Sprache sichtbar machen. Informationen visuell aufbereiten zur Unterstützung
des Verständnisses.

### Methoden
| Methode | Beschreibung | Einsatz |
|---------|-------------|---------|
| **Wortspeicher / Wortmauer** | Thematisch sortierte Fachbegriffe im Klassenraum | Dauervisualisierung, Wortschatzarbeit |
| **Satzbausteine / Satzgliederkarten** | Farbige Karten für Subjekt, Prädikat, Objekt | Satzglieder physisch verschieben/umstellen |
| **Grammatik-Farbsystem** | Nomen = blau, Verb = rot, Adjektiv = grün, Artikel = gelb | Wortarten visuell unterscheiden |
| **Textsorten-Plakate** | Merkmale und Aufbau einer Textsorte als Poster | Inhaltsangabe, Erörterung, Bericht, Brief |
| **Schreibfahrplan** | Schrittweise Anleitung zum Verfassen einer Textsorte | Geleitetes Schreiben, Prüfungsvorbereitung |
| **Mindmap / Cluster** | Brainstorming-Ergebnisse visualisieren | Themeneinstieg, Wortfeldarbeit |
| **Textmarkierung** | Farbiges Markieren von Textstellen nach Kriterien | Textanalyse: rot = Hauptaussage, grün = Beispiel |
| **Bildimpulse** | Fotos, Karikaturen, Gemälde als Schreibanlass | Beschreibung, Interpretation, Stellungnahme |
| **Lesezeichen / Lesestrategiekarten** | Laminierte Karten mit Lesestrategien | Selbstständiges Lesen unterstützen |
| **Dokumentenkamera** | Schülerarbeiten live projizieren | Schreibkonferenz, Fehleranalyse, Musterlösung |
| **Untertitelfilme** | Literaturverfilmungen mit deutschen Untertiteln | Zugang zu literarischen Stoffen |
| **Comics / Graphic Novels** | Literatur in Bildform | Leichterer Zugang, Bildbeschreibung |

### Tafelarbeit
- Stundenziel + Ablauf sichtbar
- Neue Begriffe: Wort — Erklärung — Beispiel (drei Spalten)
- Grammatikregeln: farbig hervorgehoben, mit Beispielsätzen
- Merksätze: eingerahmt, klar formuliert

### Wortschatzarbeit (besonders wichtig)
1. **Vorentlastung**: Schlüsselwörter VOR dem Lesen eines Textes klären
2. **Wort-Bild-Zuordnung**: Abstrakte Begriffe mit Bildern verbinden
3. **Kontextualisierung**: Wörter in verschiedenen Sätzen verwenden
4. **Wortfamilien**: Zusammenhänge sichtbar machen (z.B. fahren, Fahrer, Fahrt, Fahrrad)
