# Didaktik Chemie

Allgemeine fachdidaktische Referenz für den Chemieunterricht (Sekundarstufe I).

## Notation und Konventionen

### Chemische Symbolik
| Symbol | Bedeutung | Beispiel |
|--------|-----------|---------|
| → | Reaktionspfeil (reagiert zu) | Fe + S → FeS |
| + | Edukte/Produkte trennen | H₂ + Cl₂ → 2 HCl |
| ↑ | Gasförmiges Produkt (entweicht) | CaCO₃ → CaO + CO₂↑ |
| ↓ | Niederschlag (fällt aus) | AgNO₃ + NaCl → AgCl↓ + NaNO₃ |
| (s) | fest (solid) | Fe(s) |
| (l) | flüssig (liquid) | H₂O(l) |
| (g) | gasförmig (gas) | CO₂(g) |
| (aq) | in Wasser gelöst (aqueous) | NaCl(aq) |
| Δ | Wärmezufuhr (über Reaktionspfeil) | |

### Reaktionsgleichungen
Schrittweises Aufstellen (besonders für A-Kurs):
1. **Wortgleichung**: Eisen + Schwefel → Eisensulfid
2. **Formelgleichung (unausgeglichen)**: Fe + S → FeS
3. **Ausgeglichene Gleichung**: Fe + S → FeS (hier bereits ausgeglichen)
4. **Komplexeres Beispiel**: 2 H₂ + O₂ → 2 H₂O

### GHS-Gefahrensymbole

| Piktogramm | Bezeichnung | Bedeutung |
|------------|-------------|-----------|
| GHS01 | Explodierende Bombe | Explosionsgefahr |
| GHS02 | Flamme | Entzündbar |
| GHS03 | Flamme über Kreis | Brandfördernd |
| GHS04 | Gasflasche | Unter Druck stehende Gase |
| GHS05 | Ätzwirkung | Ätzend |
| GHS06 | Totenkopf | Giftig |
| GHS07 | Ausrufezeichen | Reizend/Gesundheitsschädlich |
| GHS08 | Gesundheitsgefahr | Krebserzeugend/Organschäden |
| GHS09 | Umwelt | Umweltgefährlich |

### Protokollformat
| Abschnitt | Inhalt |
|-----------|--------|
| **Versuch** | Titel (z.B. „Reaktion von Eisen mit Schwefel") |
| **Gefahrenhinweise** | GHS-Symbole + Schutzmaßnahmen (Schutzbrille, Handschuhe) |
| **Material/Geräte** | Liste mit Bildern/Skizzen |
| **Chemikalien** | Name, Formel, GHS-Symbole, Menge |
| **Durchführung** | Nummerierte Schritte mit Skizzen |
| **Beobachtung** | Was siehst/riechst du? (Tabelle: vorher → nachher) |
| **Auswertung** | Wort-/Formelgleichung, Reaktionstyp, Erklärung |
| **Ergebnis** | Merksatz |

## Typische Stundenstruktur

### Einführung (ca. 10 Min)
1. **Einstieg visuell**: Demonstrationsversuch, Alltagsgegenstand
   (z.B. verrosteter Nagel), Bild oder kurzes Video
2. **Frage/Problem**: „Was passiert hier? Warum rostet Eisen?" —
   schriftlich an der Tafel
3. **Vorwissen aktivieren**: Was wissen wir schon? (Karten/Stichpunkte)
4. **Fachbegriffe vorentlasten**: Neue Wörter an der Tafel mit Bild/Modell

### Übung (ca. 25 Min)
1. **Experiment**: Schülerversuch nach schriftlicher Anleitung
   (Arbeitsblatt mit Bildern, GHS-Symbolen, nummerierten Schritten).
   Sicherheitsregeln sichtbar aushängen.
   Beobachtungstabelle (vorher/nachher) ausfüllen.
2. **Auswertung**: Wortgleichung → Formelgleichung entwickeln.
   Teilchenmodell zeichnen oder mit Modellbaukasten nachbauen.
3. **Übung**: Reaktionsgleichungen aufstellen/ausgleichen,
   Periodensystem lesen, Stoffe zuordnen

### Sicherung (ca. 10 Min)
1. **Ergebnisse vergleichen**: Beobachtungen der Gruppen sammeln
   (Tabelle an der Tafel)
2. **Merksatz**: Gemeinsam formulieren, ins Heft übernehmen
3. **Gleichung sichern**: Reaktionsgleichung + Reaktionstyp ins Heft
4. **Hausaufgabe**: Schriftlich an der Tafel (z.B. Gleichungen ausgleichen)

## Differenzierung A-Kurs / B-Kurs

Die Fachleistungsdifferenzierung in Chemie beginnt je nach Bundesland
ab Klasse 7 oder 9 (Physik oder Chemie).

### A-Kurs (grundlegende Bildung — EBR)
- **Experimente**: Stärker angeleitet; Beobachtungstabelle vorgegeben;
  Lückentext für Auswertung
- **Reaktionsgleichungen**: Ergänzen (Edukte/Produkte teilweise vorgegeben);
  Wortgleichungen schreiben; einfache Formelgleichungen mit Hilfe
- **Periodensystem**: Informationen ablesen (Ordnungszahl, Symbol, Name);
  Hauptgruppen zuordnen
- **Fachbegriffe**: Zuordnungsaufgaben (Begriff ↔ Definition);
  Lückentext mit Wortbank
- **Berechnungen**: Einfache Stoffmengenberechnungen entfallen oder werden
  sehr vereinfacht; Fokus auf qualitatives Verständnis

### B-Kurs (erweiterte Bildung — FOR)
- **Experimente**: Offenere Aufgabenstellung; eigene Beobachtungstabelle;
  Hypothese formulieren; Fehlerbetrachtung
- **Reaktionsgleichungen**: Selbstständig aufstellen und ausgleichen;
  Reaktionstypen bestimmen und begründen
- **Periodensystem**: Zusammenhänge erkennen (Elektronenschalen → Hauptgruppe,
  Eigenschaften → Stellung im PSE)
- **Fachbegriffe**: Definitionen in eigenen Worten formulieren;
  Zusammenhänge erklären
- **Berechnungen**: Molare Masse, einfache stöchiometrische Berechnungen

### Differenzierungsbeispiel
```
Aufgabe: Reaktion von Magnesium mit Sauerstoff

A-Kurs:
  1. Ergänze die Wortgleichung:
     Magnesium + _________ → _____________
  2. Ergänze die Formelgleichung:
     ___ Mg + O₂ → ___ MgO
  3. Um welchen Reaktionstyp handelt es sich?
     □ Synthese  □ Analyse  □ Neutralisation

B-Kurs:
  1. Stelle die Reaktionsgleichung auf und gleiche sie aus.
  2. Bestimme den Reaktionstyp und begründe deine Antwort.
  3. Erkläre mithilfe des Schalenmodells, warum Magnesium
     mit Sauerstoff reagiert.
```

## Visuelle Methoden und Materialien

### Grundprinzip
Chemie spielt sich auf drei Ebenen ab — alle drei müssen visuell dargestellt werden:
1. **Makroebene** (beobachtbar): Experiment, Farbe, Form, Aggregatzustand
2. **Submikroebene** (Teilchen): Modelle, Animationen, Zeichnungen
3. **Symbolebene** (Formeln): Reaktionsgleichungen, PSE, Strukturformeln

### Methoden
| Methode | Beschreibung | Einsatz |
|---------|-------------|---------|
| **Demonstrationsexperiment** | Lehrkraft zeigt Versuch, Schüler beobachten | Einstieg, spektakuläre Reaktionen sicher zeigen |
| **Schülerexperiment** | Kleingruppen mit schriftlicher Anleitung | Handlungsorientierung, Beobachtungskompetenz |
| **Molekülbaukästen** | Kugel-Stab-Modelle zum Zusammenbauen | Teilchenvorstellung, Bindungsarten |
| **Teilchen-Theater** | Schüler spielen Atome/Moleküle; Klettverschlüsse für Bindungen | Reaktionsabläufe körperlich erfahren |
| **Periodensystem-Poster** | Großformatiges PSE an der Wand, farbig nach Gruppen | Ständige Sichtbarkeit, Nachschlagewerk |
| **Vorher-Nachher-Tabelle** | Tabelle: Stoff vorher → Beobachtung → Stoff nachher | Systematische Experimentauswertung |
| **Farbcodierung** | Stoffgruppen farbig (Metalle = grau, Nichtmetalle = gelb, Edelgase = violett) | Ordnung im PSE sichtbar |
| **Stufenmodell für Gleichungen** | Wortgleichung → Formelgleichung → ausgeglichen | Schrittweiser Aufbau der Symbolsprache |
| **Sicherheitsplakate** | GHS-Symbole + Verhaltensregeln als Poster im Labor | Dauerhafte Sichtbarkeit der Sicherheitsregeln |
| **Dokumentenkamera** | Versuchsaufbau live projizieren; kleine Reaktionen vergrößern | Alle sehen Details gleichzeitig |
| **Animationen/Simulationen** | Teilchenbewegungen, Reaktionsabläufe am Computer | Submikroebene sichtbar machen |
| **Concept Maps** | Begriffe visuell vernetzen (Stoff → Element → Atom → ...) | Zusammenhänge visualisieren |
| **Laminierte Formelkarten** | Wichtige Formeln, Symbole, Regeln | Nachschlagewerk im Unterricht |

### Tafelarbeit
- Stundenziel + Ablauf sichtbar
- Fachbegriffe: Wort — Formel — Bild/Modell (drei Spalten)
- Reaktionsgleichungen: groß, deutlich, schrittweise aufgebaut
- Merksätze eingerahmt, mit Formelbeispielen
- GHS-Symbole bei jedem Experiment an der Tafel
